Questão 7- As regras do Pré-desafio proibe os participantes de utilizar bibliotecas externas 
*Apenas recursos nativos da linguagem de programação escolhida podem ser utilizados, ou seja, bibliotecas externas não são permitidas.*
porém na pagina do desafio do cryptopals a questão nos incitiva a utilizar blibliotecas para a solução do problema. A solução dessa questão 
acaba se tornando muito complexa para implementar em Python puro.
OBS: por curiosidade, a implementação em python é significamente pior que as em C ou Rust, a perfomance sera pior
não vai ser capaz de zerar valores sensiveis na memoria alem de ser vuneravel para diversos tipos de ataque.
A questão 7 não é dificil de implementar utilizando bibliotecas, uma vez que temos que apenas utilizar as ferramentas dando a chave dada e ler 
o arquivo .txt como fizemos na questão 4. O texto decriptado é a letra da musica Play That Funky Music.
